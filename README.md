## What this explores
- Congestion under pressure
- Spontaneous alignment
- Ordered flow without global control

## Visual semantics
### Visual cues are not decorative; they expose internal system states:
- Red halos indicate individual pressure from local crowd density  
- Directional ticks show local directional agreement (order)  
- Color and radius encode speed and neighborhood density
